Clarity Kit NVDA Add On

Navigate inaccessible webpages

How to Use

NVDA + Z: press twice to activate the add on, press this button twice every time you are on an inaccessible page

NVDA + arrow keys: navigate the web page

Installation Instructions

  1. Click the download link below
  2. Press NVDA + n to open the NVDA menu
  3. Go to the tools sub menu
  4. Now go to the Add On Store
  5. Click the Install from External Source button
  6. Select select the dot nvda-addon file that you downloaded earlier
  7. Install the add on
  8. Close the store page
  9. Restart NVDA for the changes to take effect
